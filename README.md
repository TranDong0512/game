## 🚀 Chạy trên GitHub Pages

1. Fork hoặc push repo này lên GitHub
2. Vào **Settings** → **Pages**
3. Chọn branch `main`, thư mục `/ (root)`
4. Truy cập `https://username.github.io/repo-name/`

## 💻 Chạy local

Mở file `index.html` trực tiếp trong trình duyệt, hoặc:

```bash
# Python
python -m http.server 8000

# Node.js
npx serve .
```
